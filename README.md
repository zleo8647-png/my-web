# 张小龙 · 个人作品集网站

> 资深互联网软件产品经理 · 12年全链路产品专家

一个基于纯 HTML/CSS/JS 构建的个人作品集主页，Apple 风格暗色设计，展示产品经理的核心能力、工作经历与数据成果。

## 项目预览

页面包含以下模块：

- **Hero 区** — 个人介绍与核心标签
- **产品理念** — 职业价值观陈述
- **个人优势** — 四大核心竞争力卡片
- **技能矩阵** — 核心技能 + 工具链标签云
- **工作经历** — 四段完整工作经历（含图表与动态数据）
- **数据成果大屏** — 关键业绩指标看板
- **教育经历** — 中山大学计算机应用技术本科
- **联系方式** — 电话 / 邮箱

## 文件结构

```
├── index.html   # 主页（所有样式与逻辑内联）
└── tx.png       # 头像图片（需自行准备）
```

## 使用方法

无需任何构建工具或依赖安装，直接用浏览器打开即可：

```bash
# 方法一：直接双击 index.html 用浏览器打开

# 方法二：用 VS Code Live Server 插件预览
# 右键 index.html → Open with Live Server
```

> ⚠️ 头像图片路径为 `tx.png`，请将你的头像文件命名为 `tx.png` 并放在 `index.html` 同级目录下。

## 技术栈

- **HTML5 / CSS3** — 原生实现，无框架依赖
- **JavaScript** — 原生 JS，滚动动画 + 数字计数动效
- **Chart.js 4.4.0** — 漏斗对比图 + 折线趋势图（CDN 引入）
- **CSS 变量** — 统一设计 Token，便于主题定制

## 设计风格

- 暗色系（黑底 + 荧光绿 `#2AF740` 点缀）
- Apple 风格滚动渐入动画（`IntersectionObserver`）
- 响应式布局，适配手机（480px）、平板（768px）与桌面端

## 自定义修改

| 需要修改的内容 | 对应位置 |
|---|---|
| 姓名 / 职位 / 简介 | `<section class="hero">` |
| 工作经历 | `<section id="experience">` |
| 联系方式（电话/邮箱） | `<div class="cta-section" id="contact">` |
| 主题色 | CSS 变量 `--green: #2AF740` |
| 头像图片 | 替换 `tx.png` 文件 |
